# Smart Farming ESP32 + Nano

## Fitur

- Monitoring suhu, kelembaban, pH, cahaya
- Penyiraman otomatis
- Pemupukan otomatis
- Web control
- Integrasi InfluxDB

## Setup

1. Isi WiFi & token
2. Upload ke ESP32

## Arsitektur

ESP32 sebagai master, Arduino Nano sebagai sensor node via RS485
